Install Guide

怎么装到桌面

iPhone

用 Safari 打开本页，点“分享”，再点“添加到主屏幕”。

Android

用 Chrome 打开后可点“安装到桌面”，或从浏览器菜单里选择“安装应用”。

数据说明

这是手机浏览器本地版，API 配置与结果缓存只保存在当前设备浏览器中。

API 设置

推荐给手机端单独准备一个 API Key。PWA 无法直接访问桌面端本地数据库和共享目录。
